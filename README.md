-AI Interviewer Platform

An AI-powered interview simulation platform built using the MERN stack that helps candidates practice interviews and enables recruiters to assess candidates intelligently using AI-driven insights.

-Overview

The AI Interviewer Platform simulates real-world interview scenarios by:

Taking candidate resumes as input

Asking AI-generated interview questions

Recording and analyzing candidate responses

Generating performance reports using AI

This platform is designed to bridge the gap between interview preparation and real interview experience.
